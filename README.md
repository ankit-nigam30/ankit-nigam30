Technical Lead with hands-on expertise in AWS cloud architecture, Linux systems, and automation, delivering secure, scalable, and cost-optimized cloud solutions for enterprise environments. Experienced in designing and managing AWS infrastructure using services such as EC2, VPC, IAM, S3, RDS, Lambda, CloudFormation, Route 53, ELB, CloudWatch, and CloudTrail.

Strong background in Linux (Amazon Linux, Ubuntu), with deep knowledge of system administration, networking, and shell scripting. Proficient in Python and Bash automation to improve reliability, operational efficiency, and cloud governance.

Brings a solid architecture and DevOps mindset, with exposure to CI/CD, infrastructure as code, monitoring, and Git-based collaboration. Known for client-facing communication, technical leadership, and cross-team collaboration in global delivery environments. Passionate about building high-availability, secure cloud architectures and supporting onsite and international cloud transformation initiatives.
